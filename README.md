# github-finder

Search any GitHub username and view their profile, stats, and public repositories.

**Live Demo:** [your-vercel-url-here]

---

## Features

- Search GitHub users by username
- View profile info, bio, and stats
- Browse public repositories with stars, forks, and language
- Clean, minimal interface inspired by GitHub's own design

## Tech Stack

- React (Vite)
- GitHub REST API
- CSS

## Run Locally

```bash
git clone https://github.com/rohit-dhok/github-finder.git
cd github-finder
npm install
npm run dev
```
